This application allows users to take control of their fitness journey by moving away from pen-and-paper logs to a digital dashboard. Users can browse a vast database of exercises, create custom workout logs, and visualize their strength progression over time.

The project is built as a responsive Single Page Application (SPA), ensuring a seamless experience across desktop, tablet, and mobile devices.
